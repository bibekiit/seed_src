DESCRIPTION 

------------------------------------------------------------------------
orientation_image_rao
Computes the orientation image for a fingerprints. 
Usage:
[gimg,oimg] = orientation_image_rao(x)
oimg - orientation image
gmag - gradient magnitude
x    - fingerprint image 
NOTE: The image x dimension should be multiple of 8
